SIM210: Use 'bool(a)' instead of 'True if a else False'

It reduces cognitive complexity slightly.

Complex Form:

True if a else False


Simplified From:

bool(a)


Rationale:
Both notations are equivalent in Python 3, but the second one is shorter.
