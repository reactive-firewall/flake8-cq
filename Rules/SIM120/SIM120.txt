SIM120: Subclassing object is implied in modern Python.

Use 'class FooBar:' instead of 'class FooBar(object):'

Complex Form:

class FooBar(object):
    ...


Simplified From:

class FooBar:
    ...

Rationale:
Both notations are equivalent in Python 3, but the second one is shorter.
