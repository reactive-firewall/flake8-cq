# SIM202: Use 'a == b' instead of 'not a != b'

Simplify the negation of equivalence. It reduces cognitive complexity.

## Complex Form

```python
not a != b
```

## Simplified From

```python
a == b
```

## Rationale

Both notations are equivalent in Python 3, but the second one is shorter.
